# 폴더구조
## pages폴더
nextjs pages 폴더는 라우터 기능을한다.
index는 각 폴더 명이 주소가 된다.
나머지는 이름명시한 파일이 라우터 경로가 된다.
## src 폴더
나머지 모든 컴포넌트가 들어가는 폴더
추후에 사용할 typesciprt를 사용하여 tpye을 명시한 파일이 들어가는 폴더
### components 폴더
실제로 사용하는 모든 페이지들이 작성되는 폴더
### UI폴더
Storybook을 이용하여 UI를 만드는 폴더
